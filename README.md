## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/laurendayle/js-calculator
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
